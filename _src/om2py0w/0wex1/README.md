## 交互式日记文件V0.1

### 使用说明
1. 运行 main.py
2. 根据提示输入日记名，直接按ENTER表示默认diarybook作为文件名
3. 读之前的文件
4. 写文件，退出直接按ENTER

###　设想我们如何写日记:
1. 动因:想记录一些东西在日记中
2. 寻找日记本,笔和必要参考资料
3. 打开笔记本
4. 阅读已经有的内容,并找到相接的空白处
5. 写下所想的东西
6. 翻阅参考资料,将记录完善
7. 写下索引和日期,以便之后寻找和阅读
8. 关上日记,放在固定的位置以便以后再用
